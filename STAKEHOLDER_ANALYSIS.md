# Stakeholder Analysis – Aura Reels Movie Rental System

| Stakeholder          | Role                                                       | Key Concerns                               | Pain Points                         | Success Metrics                              |
| -------------------- | ---------------------------------------------------------- | ------------------------------------------ | ----------------------------------- | -------------------------------------------- |
| Customers            | Users who browse and rent movies from the system           | Easy movie discovery, quick rental process | Difficulty finding movies quickly   | Rent a movie within 2 minutes                |
| Staff                | Manage movie catalog and assist users                      | Easy management of movie data              | Manual editing of movie information | Update movie listings quickly                |
| Administrators       | Oversee the entire platform and system usage               | System reliability and user management     | Managing many users manually        | System runs smoothly with minimal errors     |
| System Administrator | Responsible for deployment, server maintenance, and uptime | Security, uptime, and stability            | System crashes or server downtime   | Maintain 99% system uptime                   |
| Developers           | Maintain and update the system                             | Clean code and clear documentation         | Hard-to-maintain systems            | Easy updates and feature additions           |
| Business Owners      | Ensure the system attracts users and generates engagement  | User satisfaction and platform growth      | Low engagement                      | Increased number of rentals and active users |
