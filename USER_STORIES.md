# 🎬 User Stories – Aura Reels Movie Rental System

---

## User Stories Table

| Story ID | User Story | Acceptance Criteria | Priority |
|----------|-----------|--------------------|----------|
| US-001 | As a user, I want to register an account so that I can access system features. | User must enter valid email and password; account created successfully. | High |
| US-002 | As a user, I want to log in so that I can access my account. | Valid credentials allow login; invalid credentials show error. | High |
| US-003 | As a user, I want to browse movies so that I can see available options. | Movies displayed with title, genre, rating. | High |
| US-004 | As a user, I want to search for movies so that I can quickly find specific titles. | Results displayed within ≤2 seconds; relevant matches shown. | High |
| US-005 | As a user, I want to view movie details so that I can decide what to rent. | Details include title, description, genre, rating. | High |
| US-006 | As a user, I want to add movies to a cart so that I can rent multiple movies. | Selected movies appear in cart immediately. | High |
| US-007 | As a user, I want to rent movies so that I can access content. | Rental is processed; confirmation is sent. | High |
| US-008 | As a user, I want to view my rental history so that I can track my rentals. | Rental history displayed correctly. | Medium |
| US-009 | As an admin, I want to manage movie catalog so that I can keep content updated. | Movies can be added, edited, deleted. | High |
| US-010 | As an admin, I want to manage users so that I can control system access. | Admin can view and manage users. | Medium |
| US-011 | As a business owner, I want reports so that I can monitor system performance. | Reports display user activity and rentals. | Medium |
| US-012 | As a system admin, I want user data secured so that system is safe. | Passwords encrypted; secure login enforced. | High |
