# 📊 Template Analysis and Selection – Aura Reels Movie Rental System

---

## GitHub Project Template Comparison

| Template | Columns / Workflow | Automation Features | Agile Suitability |
|----------|------------------|---------------------|-------------------|
| Basic Kanban | To Do → In Progress → Done | No automation (manual movement) | Suitable for simple workflows but limited for Agile tracking |
| Automated Kanban | To Do → In Progress → Done | Automatically moves issues based on status changes | Highly suitable for Agile as it supports continuous workflow and reduces manual effort |
| Bug Triage | New Issues → Needs Review → Closed | Automatically categorises and moves bug-related issues | Suitable for maintenance and bug tracking, not full Agile development |
| Team Planning | Backlog → Ready → In Progress → Done | Supports planning, prioritisation, and task tracking | Very suitable for Agile sprint planning and backlog management |

---

## Template Selection

The **Team Planning template** was selected for the Aura Reels Movie Rental System.

---

## Justification

The Team Planning template is the most suitable for this project because it aligns closely with Agile principles and supports structured sprint planning.

Firstly, it includes a **Backlog column**, which is essential for managing and prioritising user stories derived from Assignment 6. This allows clear organisation of tasks before they are moved into active development.

Secondly, the template supports a clear workflow from **Ready → In Progress → Done**, which matches the Agile development lifecycle. This helps track progress and ensures that tasks move logically through each stage.

Additionally, the template is flexible and allows for **customisation**, such as adding new columns like "Testing" and "Blocked". This makes it adaptable to the system’s needs and improves workflow visibility.

Compared to other templates:
- Basic Kanban lacks automation and structured backlog management
- Automated Kanban is useful but less focused on planning
- Bug Triage is limited to issue management rather than full system development

Overall, the Team Planning template best supports Agile development, sprint planning, and task management, making it the ideal choice for the Movie Rental System.
