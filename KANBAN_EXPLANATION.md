# 📋 Kanban Board Explanation – Aura Reels Movie Rental System

---

## What is a Kanban Board?

A Kanban board is a visual project management tool used to track tasks as they move through different stages of a workflow. It uses columns such as "To Do", "In Progress", and "Done" to represent the status of tasks and provides a clear overview of project progress.

---

## How the Kanban Board Visualizes Workflow

The Kanban board for the Aura Reels Movie Rental System uses multiple columns to represent different stages of development, including Ready, In Progress, Review, Testing, Blocked, and Done.

Each column represents a step in the workflow, allowing tasks (user stories) to move from one stage to another as work progresses. This provides real-time visibility into the status of each task and helps track overall system development.

---

## Work-in-Progress (WIP) Limitation

The board limits the number of tasks in progress by ensuring that only a small number of tasks are actively being worked on at a time. For example, only one or two tasks are placed in the "In Progress" column at a time.

This helps prevent overloading, improves focus, and reduces bottlenecks in the development process.

---

## Support for Agile Principles

The Kanban board supports Agile principles by enabling continuous delivery, flexibility, and adaptability. Tasks can be updated and moved dynamically as requirements change, allowing the system to evolve over time.

It also promotes transparency and accountability by clearly showing task ownership, progress, and current status. This ensures that development remains organised and aligned with project goals.
